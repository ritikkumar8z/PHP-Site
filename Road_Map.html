Basic PHP Concepts (Beginner Level)

1. Introduction to PHP

    What is PHP?

    PHP (Hypertext Preprocessor) is a popular open-source scripting language that is especially suited for web development. It can be embedded into HTML, and it allows developers to create dynamic content that interacts with databases. PHP is widely used for building web applications, and its simplicity and flexibility make it an excellent choice for both beginners and experienced developers.

    PHP scripts are executed on the server, and the result is sent to the client as plain HTML. This makes PHP a server-side language, meaning it is executed on the server before the page is sent to the user's web browser.

    With PHP, you can perform various tasks such as:

    • Handling forms and user input
    • Sending and receiving cookies
    • Connecting to and managing databases
    • Creating session management systems
    • Generating dynamic page content

    Overall, PHP is a versatile language that powers a significant portion of the web.

    Installing PHP (Local Development Environment with XAMPP/WAMP or using Docker)

    To start using PHP, you can install it in a local development environment using tools like XAMPP or WAMP, or you can set it up in a container using Docker. These tools provide an easy way to set up a web server and database on your local machine.

    Running a Simple PHP Script

    You can run PHP scripts by saving them with a .php extension and accessing them through a web server. For example:

                <?php
            echo "Hello, World!";
            ?>

    Embedding PHP in HTML

    PHP code can be embedded directly within HTML by using the <?php ... ?> tags. For example:

                <html>
            <body>
              <?php
              echo "This is embedded PHP!";
              ?>
            </body>
            </html>

2. PHP Syntax and Fundamentals

    PHP Tags: <?php ... ?>

    PHP code is enclosed in special tags. The standard tag for PHP code is <?php ... ?>. You can also use <? ... ?> for short tags, but it's not recommended.

    Echo vs Print

    echo is a language construct and can take multiple parameters, while print is a function and always returns 1. Example:

                <?php
            echo "Hello, World!";
            print "Hello, World!";
            ?>

    Comments in PHP (single-line, multi-line)

    Single-line comments use // or #, and multi-line comments use /* ... */.

    Variables and Data Types (String, Integer, Float, Boolean, Null)

    PHP supports several data types:

    • String: e.g., $str = "Hello";
    • Integer: e.g., $int = 42;
    • Float: e.g., $float = 3.14;
    • Boolean: e.g., $bool = true;
    • Null: e.g., $var = null;

    Constants: define()

    You can define constants using the define() function:

                <?php
            define("PI", 3.14);
            echo PI; // Outputs: 3.14
            ?>

3. PHP Operators

    Arithmetic Operators: +, -, *, /

    PHP supports the following arithmetic operators:

    • Addition (+): Adds two values. E.g., $sum = 3 + 5;
    • Subtraction (-): Subtracts one value from another. E.g., $difference = 8 - 3;
    • Multiplication (*): Multiplies two values. E.g., $product = 4 * 2;
    • Division (/): Divides one value by another. E.g., $quotient = 10 / 2;

    Assignment Operators: =, +=, -=

    Assignment operators assign values to variables:

    • = : Assigns a value to a variable. E.g., $x = 5;
    • += : Adds a value to a variable. E.g., $x += 2; // Same as $x = $x + 2;
    • -= : Subtracts a value from a variable. E.g., $x -= 3; // Same as $x = $x - 3;

    Comparison Operators: ==, ===, !=, <>

    Comparison operators compare two values:

    • == : Equal to. E.g., $x == $y; (Returns true if values are equal)
    • === : Identical to. E.g., $x === $y; (Returns true if values and types are equal)
    • != : Not equal to. E.g., $x != $y; (Returns true if values are not equal)
    • <> : Not equal to. Equivalent to !=.

    Logical Operators: &&, ||, !

    Logical operators combine conditional statements:

    • && : Logical AND. E.g., $x && $y; (Returns true if both conditions are true)
    • || : Logical OR. E.g., $x || $y; (Returns true if either condition is true)
    • ! : Logical NOT. E.g., !$x; (Returns true if the condition is false)

    Increment/Decrement Operators: ++, --

    Increment and Decrement operators modify the value of a variable by 1:

    • ++ : Increment. E.g., $x++; (Adds 1 to the variable)
    • -- : Decrement. E.g., $x--; (Subtracts 1 from the variable)
